## 🛠️ Tools yang Digunakan

- **XAMPP** — Web server lokal (Apache + MySQL)
- **WordPress** — Platform CMS
- **WooCommerce** — Plugin e-commerce (gratis)
- **Elementor Free** — Page builder visual (gratis)
- **phpMyAdmin** — Manajemen database

---

## 📋 Prasyarat

Sebelum memulai praktikum, pastikan:
- Komputer dengan RAM minimal 4GB
- Windows 10/11 (atau macOS/Linux)
- Koneksi internet untuk download XAMPP dan WordPress
- Data produk dari UMKM mitra (foto, harga, deskripsi)
